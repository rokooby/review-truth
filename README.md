# Review Truth Verification

GenLayer smart contract for verifying product reviews authenticity using AI analysis.

- **Contract**: `backend/review-truth.py`
- **App**: https://rokooby.github.io/review-truth/

## Features

- AI-powered review authenticity verification
- Fake review detection on blockchain
- Immutable review records
- Trust score calculation

## Tech Stack

- **Backend**: GenLayer (Python)
- **Frontend**: React + TypeScript + Vite
- **Blockchain**: GenLayer Network
